# **App Name**: VoiceAttach

## Core Features:

- Voice Recording: Enable users to record audio directly within the application using the device's microphone.
- Audio Download: Allow users to download the recorded audio in a common format (e.g., .mp3).
- Email Attachment: Provide a feature to attach the downloaded audio file to an email.

## Style Guidelines:

- Primary color: A soft blue (#A0BFE0) to evoke calmness and clarity.
- Background color: Light grey (#F0F0F0) for a clean and unobtrusive backdrop.
- Accent color: Teal (#468080) for interactive elements like buttons and links, providing a touch of vibrancy and indication of tappable features.
- Font: 'PT Sans', a humanist sans-serif, will be used for both headlines and body text to ensure readability and a modern aesthetic.
- Use clear, simple icons to represent actions like recording, downloading, and attaching files. Icons should be monochrome for consistency.
- A clean, straightforward layout with a prominent record button, followed by download and email options. Utilize a single-column design for ease of use.
- Subtle animations to indicate recording status and download progress, providing users with clear feedback.
